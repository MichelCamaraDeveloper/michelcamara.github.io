Proposta - Novo app CBN Amazônia em React Native
Anterior Próximo

Desenvolvi essa proposta de novo aplicativo, feita com React Native, para a rádio CBN Amazônia, enquanto trabalhei no Grupo Rede Amazônica.

Eu era responsável pelo site e pelo já publicado aplicativo, mas eu queria propor algo novo ao app, apresentei essa ideia ao meu ex-gerente mas ele não aprovou.

Telas

  1. Home - Player da rádio online, Noticias destaques do dia, últimas publicações de colunistas e programas.
  2. Noticias - Todas notícias referente a cidade escolhida.
  3. Ao vivo Rádio - Usuário escolhe a transmissão de qual cidade quer ouvir.
  4. Ao vivo Estúdio - Stream ao vivo do Estúdio CBN Manaus.
  5. Comentáristas - Exibição de todos os comentáristas, ao clicar na imagem do escolhido, era listado todas suas publicações.
  6. Programas - Exibição de todos os programas, ao clicar na imagem do escolhido, era listado todos seus "episódios".
  7. Sobre - Informações de contato com a CBN Amazônia e Informações de créditos direitos.

Etapas de Desenvolvimento

  1. Back-end - API em NodeJS.
  2. Front-end Mobile - Desenvolvido em React Native.

Poké Finder
Anterior Próximo

Esse projeto é uma iniciativa pessoal para treinar os conhecimentos sobre o framework ReactJS e React Native.

Esses sistema consome dados de uma API de pokémons e os exibe organizadamente para o usuário, ao lado esquerda da tela há um filtro que permite escolher qual o tipo do pokémon deve ser listado.

É simples, mas pude reforçar aprendizados anteriores, como por exemplo, passagem de dados por componentes, eventos, estados, hooks, funções assincronas, estilos(css), responsividade, manipulação de dados.
